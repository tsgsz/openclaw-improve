# ETF 基本面分析技能

> 适用范围：A股/港股/美股 ETF 筛选与分析
> 分析目标：跟踪效率、成本结构、流动性、风险识别
> 非技术分析：不预测价格走势，专注产品质地评估

**重要声明**：以下流程仅用于信息整理与风险识别，不构成任何投资建议或收益承诺。ETF存在跟踪误差、折溢价、流动性等风险，请自行决策并承担风险。

## 0. 统一输出规范（Telegram 友好）

**每次输出必须包含：**
- 数据更新时间（Asia/Shanghai）：`YYYY-MM-DD HH:mm`
- 数据来源与口径说明
- 筛选条件（硬条件/软条件）
- 风险提示（至少 3 条）

**展示格式：**
- 用列表，避免大表格
- 每只ETF：**1 行摘要 + 3~5 条要点**
- 结尾给"下一步核验清单"

**强制字段（每只ETF必须有）：**
- **标的指数**：跟踪什么指数，指数性质（宽基/行业/主题/跨境/商品）
- **费率一句话**：管理费+托管费合计，与同类比较
- **流动性一句话**：规模、日均成交额、折溢价状态
- **风险一句话**：该ETF类型的特有风险提示

> 若信息拿不到：必须显式标注 **`[未核验]`**

## 1) 接单澄清（必须问/必须确认）

当用户说"分析某ETF"或"筛选ETF"时，先明确以下问题：

### 基础信息
1. **市场范围**：A股/港股/美股/多市场？
2. **ETF类型偏好/回避**：
   - 宽基（沪深300/标普500等）
   - 行业/主题（科技/医药/新能源等）
   - 跨境（纳指/恒科等）
   - 商品/黄金
3. **是否回避特定类型**：高溢价QDII、期货型商品ETF等？

### 筛选条件
4. **规模门槛**：最小AUM要求？（建议A股≥2亿，美股≥$100M）
5. **费率上限**：可接受最高费率？（宽基建议≤0.5%，行业≤0.8%）
6. **折溢价容忍度**：最大可接受溢价？（建议≤2%，跨境≤3%）
7. **跟踪误差容忍度**：建议年化跟踪误差≤2%

### 使用场景
8. **投资周期**：长期持有 vs 短期交易（影响流动性要求）
9. **税务考虑**：是否需要税收效率高的结构？

## 2) 数据源选择与优先级

### 2.1 一级数据源（官方/权威）

| 数据类型 | A股来源 | 美股来源 | 港股来源 |
|---------|--------|---------|---------|
| **基金公告/招募书** | 巨潮资讯网 | SEC EDGAR | 港交所披露易 |
| **净值/持仓** | 基金公司官网 | Fund Website | 基金公司官网 |
| **实时行情** | 交易所行情 | NYSE/NASDAQ | 港交所 |
| **折溢价率** | 交易所/基金公司 | Bloomberg | 港交所 |

### 2.2 二级数据源（交叉验证）

| 数据类型 | 推荐来源 |
|---------|---------|
| **综合数据** | 理杏仁、且慢、雪球 |
| **跟踪误差** | 基金定期报告、Morningstar |
| **行业对比** | Wind、同花顺iFinD |
| **国际数据** | ETF.com, ETF Database, JustETF |

### 2.3 交叉验证策略

- **净值 vs 交易价格**：至少2个来源核对，差异明显时以交易所官方为准
- **费率**：以招募说明书为准，行情网站可能滞后
- **折溢价**：注意时差（跨境ETF的NAV和交易价格时间可能不匹配）
- **跟踪误差**：以基金年报/季报披露为准

## 3) 核心指标口径

### 3.1 费率结构
```
总费率 = 管理费 + 托管费 + 其他费用（销售服务费、指数使用费等）

- A股通常披露：管理费 + 托管费
- 美股通常披露：Expense Ratio（总费率）

参考标准：
- 宽基ETF：≤0.20%（优秀），≤0.50%（可接受）
- 行业ETF：≤0.50%（优秀），≤0.80%（可接受）
- 跨境/QDII：≤0.60%（优秀），≤1.00%（可接受）
- 商品/黄金：≤0.40%（优秀），≤0.60%（可接受）
```

### 3.2 跟踪误差与跟踪偏离度
```
跟踪偏离度（Tracking Difference）= ETF累计收益率 - 指数累计收益率
跟踪误差（Tracking Error）= 跟踪偏离度的年化标准差

参考标准：
- 优秀：年化跟踪误差 ≤0.50%
- 可接受：年化跟踪误差 ≤2.00%
- 警惕：年化跟踪误差 >3.00%

注意：费率是跟踪误差的主要来源，高费率ETF跟踪误差通常更大
```

### 3.3 规模与流动性
```
规模（AUM）：
- A股：≥2亿人民币（避免清盘风险）
- 美股：≥$100M（避免清盘风险）
- 港股：≥1亿港元

流动性指标：
- 日均成交额：建议≥1000万（A股）/≥$1M（美股）
- 买卖价差（Bid-Ask Spread）：≤0.20%（优秀）
```

### 3.4 折溢价率
```
折溢价率 = (ETF交易价格 - NAV) / NAV × 100%

参考标准：
- 正常区间：-0.5% ~ +0.5%
- 可接受：-2% ~ +2%
- 警惕：
  - 普通ETF：溢价>3%或折价<-3%
  - QDII ETF：溢价>5%（额度限制导致套利失灵）

⚠️ 特别注意：跨境ETF需确认NAV计算时间与交易时间是否匹配
```

## 4) ETF 类型差异化分析框架

### 4.1 宽基ETF分析清单

**核心关注点**：
- [ ] **指数编制方法**：市值加权/等权/因子加权/策略加权
- [ ] **行业偏离度**：与基准指数的行业配置对比
- [ ] **成分股权重集中度**：前10大权重股占比
- [ ] **调仓规则**：调仓频率、冲击成本

**指标参考**：
| 指标 | 优秀 | 可接受 | 警惕 |
|-----|-----|-------|-----|
| 前10大权重占比 | ≤40% | ≤60% | >70% |
| 年化跟踪误差 | ≤0.5% | ≤2% | >3% |
| 费率 | ≤0.2% | ≤0.5% | >0.8% |

**常见宽基对比**：
| 指数 | 市场 | 加权方式 | 特点 |
|-----|-----|---------|-----|
| 沪深300 | A股 | 自由流通市值加权 | 大盘蓝筹代表 |
| 中证500 | A股 | 自由流通市值加权 | 中小盘成长 |
| 标普500 | 美股 | 市值加权（委员会选择） | 质量因子隐含 |
| 纳指100 | 美股 | 市值加权 | 科技集中度高 |
| 恒生指数 | 港股 | 市值加权 | 金融地产占比高 |

---

### 4.2 行业/主题ETF分析清单

**核心关注点**：
- [ ] **集中度风险**：单一行业占比是否过高
- [ ] **政策敏感度**：行业是否受政策调控（医药集采、教培、新能源补贴等）
- [ ] **估值合理性**：行业PE/PB与历史分位对比
- [ ] **主题纯度**：是否有大量非相关业务混入
- [ ] **竞争格局**：同类主题ETF的数量与费率对比

**风险识别模板**：
```
【行业ETF风险扫描】
1. 集中度风险：该行业占组合___%，前3大个股占行业___%
2. 政策风险：【高/中/低】- 近期政策变化：___
3. 估值位置：PE(TTM)___，历史分位___%，判断：【低估/合理/高估】
4. 同类对比：市场共有___只同类ETF，该ETF费率排名___/___
```

---

### 4.3 跨境ETF分析清单

**核心关注点**：
- [ ] **汇率风险**：是否对冲？历史汇率波动影响
- [ ] **QDII额度状态**（A股QDII ETF）：额度是否充足？是否暂停申购？
- [ ] **时差交易风险**：基础市场休市时，ETF成为"价格发现工具"
- [ ] **折溢价异常**：跨境ETF折溢价可能持续扩大
- [ ] **税务处理**：预扣税、资本利得税

**QDII ETF 特殊监控**：
```
【QDII ETF 溢价风险警报】
触发条件（任一）：
- 折溢价率 > 5%
- 基金公司公告暂停申购
- 市场传闻QDII额度紧张

应对：
- 暂停买入，等待溢价回归
- 寻找替代标的（如港股通ETF）
```

**参考来源**：
- [Invesco ETF套利机制研究](https://www.invesco.com/apac/en/institutional/insights/etf/understanding-etf-trading-and-liquidity-etf-arbitrage-premiums-and-discounts.html)
- 国家外汇管理局QDII额度公示

---

### 4.4 商品/黄金ETF分析清单

**核心关注点**：
- [ ] **持仓结构**：实物持仓 vs 期货合约
- [ ] **期货曲线结构**（期货型）：Contango vs Backwardation
- [ ] **Roll Cost/收益**：展期损耗或收益
- [ ] **仓储成本**（实物型）：对净值的影响
- [ ] **替代工具对比**：期货、实物、ETF、相关股票

**期货曲线分析**：
```
Contango（升水）：远期价格 > 近期价格
- 持有期货ETF会产生roll cost（展期损耗）
- 年化损耗可能高达10%+

Backwardation（贴水）：远期价格 < 近期价格
- 持有期货ETF会产生roll yield（展期收益）

实物黄金ETF：
- 无roll cost问题
- 但需支付仓储费（已计入费率）
```

**参考来源**：
- [Fidelity - Commodity ETFs Contango](https://www.fidelity.com/learning-center/investment-products/etf/commodity-etfs-contango-backwardation)
- [Investopedia - Contango](https://www.investopedia.com/terms/c/contango.asp)

## 5) 风险清单（价值陷阱识别）

### 5.1 高优先级风险（立即规避）

| 风险 | 识别方法 | 后果 | 应对 |
|-----|---------|-----|-----|
| **高溢价接盘** | 折溢价率>5%（QDII）或>3%（普通） | 溢价回归时直接亏损 | 暂停买入，等溢价回归 |
| **规模过小清盘** | AUM<清盘线（A股约2亿） | 被迫赎回、投资中断 | 选择规模更大的同类产品 |
| **极端跟踪误差** | 年化跟踪误差>3% | 无法有效跟踪指数 | 更换管理人 |
| **流动性枯竭** | 日均成交额<100万 | 无法以合理价格成交 | 选择流动性更好的产品 |

### 5.2 中优先级风险（需谨慎）

| 风险 | 识别方法 | 应对 |
|-----|---------|-----|
| **行业集中度** | 单一行业>80% | 控制仓位，分散配置 |
| **政策黑天鹅** | 监管政策突变 | 关注政策动向，设止损 |
| **Contango损耗** | 期货型商品ETF | 避免长期持有，短期交易 |
| **高费率侵蚀** | 同类产品中费率最高 | 选择低费率替代品 |

### 5.3 低优先级风险（了解即可）

- 调仓冲击成本（高频调仓指数）
- 税务效率差异
- 做市商退出风险
- 指数编制规则变更

## 6) 标准工作流

### 6.1 单只ETF深度分析流程

1. **信息收集**
   - 获取基金基本信息（类型、跟踪指数、成立日期）
   - 获取最新净值、价格、折溢价率
   - 获取费率结构、规模、成交额

2. **定量分析**
   - 计算/查询跟踪误差
   - 对比同类ETF费率
   - 评估流动性指标

3. **定性分析**
   - 根据ETF类型执行差异化检查清单
   - 识别特有风险
   - 评估替代产品

4. **交叉验证**
   - 至少2个数据源核对关键指标
   - 查阅最新公告（暂停申购、费率调整等）

5. **输出报告**
   - 按Telegram友好格式输出
   - 包含风险提示和下一步核验清单

### 6.2 ETF筛选流程

1. **明确筛选条件**（第1节）
2. **获取候选池**：全市场ETF列表或特定类型列表
3. **硬条件过滤**：
   - 规模≥门槛
   - 费率≤上限
   - 折溢价在正常区间
4. **排序打分**：
   - 费率权重30%
   - 跟踪误差权重30%
   - 流动性权重20%
   - 规模权重20%
5. **人工复核**：对Top N只进行差异化分析
6. **输出结果**：按Telegram友好格式
